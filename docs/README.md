## 📁 Estrutura de Pastas

```
alternativa-perfeita/
│
├── index.html              ← Página principal (HTML puro, sem frameworks)
│
├── css/
│   ├── variables.css       ← Variáveis CSS (cores, tokens de design)
│   ├── global.css          ← Componentes globais: nav, botões, seção, WhatsApp
│   ├── sections.css        ← Estilos de cada seção da página
│   └── responsive.css      ← Media queries (1024px, 768px, 480px)
│
├── js/
│   ├── particles.js        ← Animação de partículas no canvas
│   ├── nav.js              ← Menu mobile (hamburger)
│   ├── scroll-reveal.js    ← Animação de entrada dos elementos
│   ├── lightbox.js         ← Visualizador de imagens da galeria
│   ├── carousel.js         ← Carrossel automático de depoimentos
│   └── contact.js          ← Formulário → envio via WhatsApp
│
├── images/
│   └── README.md           ← Instruções para adicionar imagens próprias
│
└── docs/
    ├── README.md           ← Este arquivo
    ├── CUSTOMIZACAO.md     ← Como personalizar cores, textos e seções
    └── DEPLOY.md           ← Como publicar o site online
```

---

## 🚀 Como Abrir o Site

1. Descompacte o arquivo `.zip`
2. Abra a pasta `alternativa-perfeita/`
3. Dê duplo clique em **`index.html`**
4. O site abrirá no seu navegador padrão

> ⚠️ Para que o mapa do Google funcione corretamente, o site precisa ser
> servido via HTTP (não apenas aberto como arquivo local). Veja `docs/DEPLOY.md`.

## 🛠️ Tecnologias Utilizadas

- **HTML5** semântico
- **CSS3** com variáveis nativas, Grid e Flexbox
- **JavaScript** vanilla (sem dependências externas)
- **Google Fonts** — Bebas Neue + Barlow + Barlow Condensed
- **Unsplash** — imagens de placeholder (substituir por fotos reais)
- **Google Maps** Embed API
