## Como referenciar no código

Após salvar as imagens aqui, substitua no `index.html` e `css/sections.css`:

```
url('https://images.unsplash.com/...')
      ↓
url('images/nome-do-arquivo.jpg')
```

## Formatos aceitos
- `.jpg` / `.jpeg` — recomendado para fotos (menor tamanho)
- `.png` — recomendado para logos e imagens com transparência
- `.webp` — melhor compressão (suporte moderno)
